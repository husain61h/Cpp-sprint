Rock Paper Scissors (C++)

A simple console based Rock Paper Scissors game written in C++. The user plays against the computer for a chosen number of rounds and a final scoreboard is shown at the end.

Features

- User vs Computer gameplay

- Random computer move generation

- Input validation

- Score tracking (wins, losses, draws)

- Object oriented design using a class

Concepts Used

- Classes and functions

- Random numbers

- Input validation

- Formatted output
