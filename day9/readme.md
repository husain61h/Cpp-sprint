Bank Account Program is a simple C++ program that demonstrates basic object oriented programming concepts like classes and inheritance using a bank account example.

Features

- Create a bank account with a name and initial balance

- Deposit money into the account

- Withdraw money from a savings account with balance check

- Display the final balance

Concepts Used

- Classes and objects

- Inheritance (SavingAccount inherits from BankAccount)

- User input and output

- Basic validation for withdrawals
