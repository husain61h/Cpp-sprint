The Smart Expense Analyser provides a set of practical features to help users better understand their spending habits through simple input and clear analysis. 
It focuses on ease of use, clean data handling, and meaningful insights.

- Records multiple expense transactions along with their categories

- Detects and flags high-value expenses at the time of entry

- Automatically formats expense category names for consistency and readability

- Calculates the total expense across all transactions

- Computes the average expense per transaction

- Displays a detailed category wise breakdown of expenses

- Identifies the highest spending category, including ties when applicable

- Analyzes spending patterns and classifies them as controlled, moderate, or high spending

- Presents results using well aligned and easy to read console output

- Demonstrates effective use of C++ concepts such as classes, vectors, maps, and string manipulation
