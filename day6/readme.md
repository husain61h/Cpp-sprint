Vault OS 1.0 is a console-based credential management system written in C++.
It allows users to securely store, view, update, and remove login credentials using a JSON file for persistent storage.

This project focuses on practicing file handling, input validation, and basic object-oriented programming concepts in C++.

Features

- Store website credentials (website, username, password)

- View all saved credentials

- Update an existing password

- Remove saved credentials

- Persistent storage using a JSON file (vault.json)

- Input validation to prevent crashes

- Clean menu-driven interface with screen clearing

- Automatic database initialization on first run

Requirements

- C++17 or later

- json.hpp (nlohmann JSON library)

- Windows terminal (uses system("cls") for screen clearing)
