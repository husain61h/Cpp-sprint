Student Result Program is simple C++ program demonstrating basic Object Oriented Programming using a `Student` class.

Features
- Stores student name, roll number, and marks
- Uses a constructor for initialization
- Displays student details
- Checks pass/fail based on marks

How It Works
- Takes student details as input
- Creates a `Student` object
- Displays information and result

Concepts Used
- Classes and Objects
- Constructors
- Member functions
- Conditional statements
